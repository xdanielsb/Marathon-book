# Marathon's book
---


Configurations and ways to fire up the scripts

## Go

```sh
$ export PATH=$PATH:/opt/go/bin
$ export GOPATH=$HOME/Documents/golang
$ export GOROOT=/opt/go

$ go run filename.go

```

## Java

```sh
$ export JAVA_HOME=/usr/local/jdk1.8.0_111
$ export PATH=$JAVA_HOME/bin:$PATH


$ javac filename.java
$ java filename
```
